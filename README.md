## Bad Apple in Ableton
Remote Scripts in Ableton are great.  
It opens you to create some pretty neat integration with the DAW Ableton.

Clips colors can be changed using Remote Scripts.  
I've always had this silly idea to play the video Bad Apple on Ableton Clips, so I did it.  

#### [Preview](https://youtu.be/4pBRVbXzuKg?t=27)

### Install
Download the ZIP and extract it in the right location:
Windows: `C:\Users\[username]\Documents\Ableton\User Library\AbletonBadApple`  
MacOS: `Macintosh HD/Users/[username]/Music/Ableton/User Library/AbletonBadApple`

Issues:
- It can be laggy if there are too many clips to change colors. The solution kinda gives a "sped up" kind of effect.
